# 목적
- Framework의 도움 없이 간단한 유저의 생성, 및 삭제 관련 router 생성 및 처리
- No DB Depends
- 미들웨어 패턴으로, 에러 처리 및 인증, 파싱 과정 추가

# 앱 기능

- 유저의 삭제
- 유저의 정보 변경
- 유저의 확인
